# Frontend Vanilla Boilerplate
## Supports
  * ES6
  * SCSS
  * JSON

## File tree
```
.
├── LICENSE
├── README.md
├── gulpfile.js
├── package.json
├── src
│   ├── data
│   │   └── dummy.json
│   ├── images
│   │   └── image.png
│   ├── views
│   │   └── index.html
│   ├── js
│   │   └── main.js
│   └── scss
│       ├── main.scss
│       └── base
│       │   └── _variables.scss
│       └── vendor
│           └── _normalize.scss
└── webpack.config.js
```

## Get up and running
 Run to install modules
 ```
 $ npm install
 ```

 Run to start gulp with Browsersync
 ```
 $ gulp
 ```

   __Don't edit files in the public folder!__
